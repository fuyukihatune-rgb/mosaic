# 🧩 Mosaic

DIDシステム向けの、やさしいプライベートメモアプリ（静的サイト・PWA）。
データはすべて端末の中だけに保存され、サーバーには一切送られません。

- 完全無料・オフライン動作
- インストール不要（ホーム画面に追加してアプリのように使えます）
- ファイルはこれ1セットで完結

現在のバージョン: **v1.4.0**

---

## 主な機能

- **タイムライン** … 人格を選んで、気分の絵文字といっしょに投稿
- **写真の添付** … 投稿に画像を付けられます（タップで拡大／自動で縮小して保存）
- **人格（メンバー）管理** … 名前・ID・役割・カラー・アイコン画像を設定
- **メンション・周知** … `@ID` で個別に、`@全体` でみんなに呼びかけ
- **検索** … 投稿の本文や人格名でしぼり込み
- **カレンダー** … 日付をタップして、その日の投稿をふり返り
- **統計** … 投稿数・人格ごとの割合・気分の分布
- **設定** … システム名の変更／開発者情報／寄付（BTC・ETH）／アプリの更新／データのリセット
- **プライバシーと規約** … 初回に表示。設定からいつでも読み返せます

> 💡 はじめの設定はとても簡単です。最初の人格の「名前」だけ決めればOK。
> ID（ハンドル）は空欄でもよく、その場合は自動でつきます。システム名は最初「Mosaic」で、あとから設定で変えられます。

## プライバシー

- データはすべてこの端末の `localStorage` に保存されます
- サーバーへの送信・外部通信は一切ありません
- ブラウザやアプリのデータを消すと、メモも消えます（バックアップはありません）
- このアプリは医療を目的としたものではありません

## GitHub Pages で公開する手順

### 1. リポジトリにファイルを置く
```bash
git add .
git commit -m "update"
git push origin main
```

### 2. GitHub Pages を有効化
- リポジトリ → Settings → Pages
- Source: 「Deploy from a branch」
- Branch: `main` / folder: `/ (root)` → Save

数分後、以下のURLで公開されます：
```
https://<ユーザー名>.github.io/mosaic/
```

## アップデートのしくみ

- `sw.js` がHTML本体を「ネット優先」で取得するため、公開を更新すると自動で反映されます
- 古いキャッシュは新バージョン起動時に自動で削除され、競合しません
- 既存ユーザーには、更新後に「アップデートしました」画面が一度だけ表示されます
- うまく更新されないときは、設定タブの「最新の状態にして再読み込み」ボタンで強制更新できます

バージョンを上げるときは2か所を合わせて変更します：
- `index.html` の `APP_VERSION`
- `sw.js` の `CACHE_NAME`

---

## カスタマイズ

`index.html` の先頭付近を編集するだけで変えられます：

- 開発者情報・寄付アドレス: `DEV_INFO` と `DONATIONS`
- 人格のカラーパレット: `const PALETTE = [...]`
- 気分の種類: `const MOODS = [...]`
- 全体メンションに使う言葉: `const ALL_MENTION_WORDS = [...]`
- プライバシー・規約の本文: `renderTermsBody()`（変更したら `TERMS_VERSION` を +1 すると再同意を促せます）
